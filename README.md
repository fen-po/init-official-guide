# init-official-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容不针对任何特定域名或网站，而是作为通用参考或资料展示用途。

## 项目简介

该仓库本质上是一个静态 HTML 页面集合。每个页面均独立编写，不依赖后端服务或外部数据库，可直接在浏览器中打开查看。适合存放各类说明、指南或信息页。

## 目录说明

```
.
├── index.html          # 仓库入口页面（如有）
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下为各个独立页面，命名尽量体现内容主题。
- 根目录下的 `index.html` 提供页面导航（如适用）。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部样式或脚本（或依赖已内嵌）。
- 页面内容可能包含文字、表格、图片（base64 或相对路径）等静态元素。
- 本仓库不追踪页面内容的时效性，请自行判断适用性。

## 维护说明

- 欢迎提交 Pull Request 以修正页面中的错误或补充新页面。
- 提交前请确保 HTML 文件语法正确，无明显排版问题。
- 请勿在页面中加入任何形式的推广、跟踪代码或外部链接（除非内容本身需要引用来源）。
- 仓库维护者保留对页面内容和结构的最终调整权。
